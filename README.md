## request
```
  $request = Request::all(); #取所有数据
  Request::unsetRoute()->all()  # 去除框架路由的参数 route 
  Request::unmeaning()->all();  # 不对数据进行强制转义
  Request::get('id'); # 获取单个数据 
  Request::decrypt('id')->all(); # 解密
  Request::decrypt(['id'])->all(); # 解密 多个参数 
   
```
